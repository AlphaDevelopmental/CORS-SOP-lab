## Why This Lab?

This CORS & SOP Misconfiguration Lab is part of my commitment to **hands-on security education**. I believe the best way to learn security vulnerabilities is to:

1. **Understand** the underlying concepts
2. **Exploit** real vulnerabilities in a safe environment
3. **Fix** the issues with secure code
4. **Verify** the fixes work

This lab makes all four steps practical and tangible.
